Summary

I am currently working a Scientist at LanzaTech where I develop software, analyze data, and work with experimentalists to engineer microorganisms to produce useful chemicals sustainably.

I'm fluent in Python, MongoDB, SQL, and English. I've also written a decent amount of JavaScript, HTML, CSS, Swift, and Japanese. I love to learn new technologies and use those tools to address practical needs.

I currently live in Skokie, IL and in my spare time I enjoy hiking, reading, writing, software development, and strength training.

Skills: What can I do?

Scientific Research & Data Science

  • Design experiments: I absolutely love working with experimentalists to design experiments and analyses that best answer the most pressing questions.
  • Perform statistical analyses: I have experience with both frequentist statistics as well as Bayesian statistical modeling.
  • Build machine learning models: Analyze complex datasets and make predictions. For example, I built a random forest regressor model using scikit-learn to predict chromatographic retention times in order to annotate unknown compounds in a metabolomics dataset.
  • Use cheminformatics tools: I have used tools like RDKit and mordred to process, analyze, and interpret chemical data.
  • Model biological systems: I have used Genome-scale Models (GEMs) to make predictions using cobrapy. I have also developed kinetic metabolic models, using both deterministic (e.g. ODE) and stochastic (e.g. Kinetic Monte Carlo) kinetic modeling approaches.
  • Visualize data: Data visualization is one of my favorite parts of doing science. I am most experienced with python libraries such as matplotlib and seaborn. I have also used tools like streamlit to create interactive dashboards to visualize data in user-defined ways.
  • Write scientific papers and internal reports: Clearly communicate our science to the rest of the world or company. I have been published in PLoS Computational Biology and Current Opinion in Biotechnology.
  • Give technical talks and training: Provide expertise in computational biology, data science, and metabolic engineering, which I have 5+ years of experience with.

Software Development

  • Develop research tools and APIs: For example, I helped develop the MINE Database, to unlock new modes of scientific inquiry. I have used flask to build multiple web APIs.
  • DevOps and Scripting: Create Docker images, deploy to EC2 on AWS using CodeDeploy, set security for incoming/outbound traffic in AWS, bat and bash scripting.
  • High-Performance Computing: I have used the Slurm Workload Manager to submit jobs on supercomputing clusters. For example, I used the Quest supercomputing cluster at Northwestern University to predict MS2 spectra for millions of compounds in parallel.
  • Build and query databases: I have experience building and working with MySQL (with sqlalchemy driver) as well as MongoDB (with pymongo driver) databases.
  • Contribute to frontend development: I'm familiar with vanilla JavaScript as well as basic HTML and CSS. I built this site using Jekyll, HTML, and CSS, for example.
  • Collaborate using git and GitHub: submit issues, make commits, merge changes, review and submit pull requests, organize issues into projects. I have used git and GitHub for both solo projects as well as collaborative projects.
  • Communicate and document effectively: I prefer to use numpy-style docstrings in my python code. I also implement best practices, like adding python type annotations and using descriptive variable names. I have used Sphinx to generate API documentation.
  • Develop iOS applications using SwiftUI: As a side project, I'm currently working on my first iPhone app - a crossword game for Japanese language learners (see blog for details).
